    Débanissement : bungeestaff.command.unban | /unban <Pseudo> <br /><br /> 
  
  Unités de temps : <br /><br />
  Secondes : s / secs / second / seconds <br />
  Minute : m / min / mins / minute / minutes <br />
  Heure : h / hs , hour, hours <br />
  Jours : d / ds / day / days <br /><br />
